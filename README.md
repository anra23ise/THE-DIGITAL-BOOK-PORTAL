# The Digital Book Portal
A full-stack web application designed to digitize library management and book sales. Developed as a 4th Semester Mini Project in Information Science and Engineering.

📝 Project Overview
The Digital Book Portal provides a seamless interface for users to browse, search, and purchase books while allowing administrators to manage inventory and track orders in real-time. The project follows the MVC (Model-View-Controller) architecture to ensure a clean separation between the database, business logic, and user interface.

🛠️ Tech Stack
Backend: Java, JSP (JavaServer Pages), Servlets

Frontend: HTML5, CSS3, JavaScript, Bootstrap

Database: MySQL

Server: Apache Tomcat 9.0

Build Tool: Maven

✨ Key Features
👤 User Module
Personalized Accounts: Secure login and registration.

Extensive Catalog: Browse books by title, author, or category.

Wishlist & Cart: Save books for later or manage active purchases.

Order History: View and track previous orders.

🛠️ Admin Dashboard
Inventory Management: Add, update, or remove books from the system.

Category Control: Create and manage book genres/categories.

Order Processing: Monitor and update the status of user orders.

Feedback Handling: Review user comments and suggestions.

🗄️ Database Design
The project utilizes a relational database with the following key tables:

user: Stores credentials and contact details.

book_details: Contains title, author, price, and category.

cart & order: Manages the transaction flow.

wishlist: Stores user preferences.

🚀 Getting Started
Clone the Repo: git clone https://github.com/anra23ise/THE-DIGITAL-BOOK-PORTAL.git

Database Setup: Import the provided .sql file into your MySQL Workbench.

Server Setup: Configure Apache Tomcat in your IDE (Eclipse/IntelliJ).

Run: Right-click the project -> Run As -> Run on Server.
